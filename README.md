# Entrega-3
Este trabajo integra análisis estadístico, ajuste numérico y modelado dinámico usando datos de Contolini &amp; Palkovacs (2023). Primero, se replicó la variación intraespecífica en la selección de presas (ANOVA y prueba de Tukey). Segundo, se modeló una distribución Skew-Normal y ajuste gaussiano. Finalmente, se simuló un modelo estructurado por tallas
